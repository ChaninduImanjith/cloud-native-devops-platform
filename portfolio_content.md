# Project Summary / Recruiter Description
## Cloud Native DevOps Platform

This project showcases a complete, production-grade Cloud Native DevOps Platform built from the ground up. I architected and deployed a multi-tier microservices application (React frontend, Node.js backend, PostgreSQL database) onto an Amazon EKS cluster. The infrastructure was entirely provisioned using Terraform as Infrastructure as Code (IaC). To ensure rapid and reliable software delivery, I engineered a fully automated CI/CD pipeline using GitHub Actions, pushing optimized, multi-stage Docker images to Docker Hub. 

The Kubernetes architecture implements industry best practices, including NGINX Ingress for traffic routing, Horizontal Pod Autoscaling (HPA) for dynamic load management, and Persistent Volume Claims (PVC) for stateful data. Furthermore, I integrated a robust observability stack utilizing Prometheus and Grafana for cluster monitoring and Alertmanager for incident response.

*Note: The application was successfully deployed and tested on Amazon EKS. The live AWS infrastructure has been decommissioned after project completion to manage cloud costs. All configurations remain available for immediate redeployment.*
